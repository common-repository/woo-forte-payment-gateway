== Description ==
Secure Credit Card Processing-Process Visa, MasterCard, Discover and American credit cards.
 Forte Payment Plugin supports a wide range of business types from enterprise, e-commerce, startups, NGO, retail, insurance and many more.
Forte Payment Plugin is PCI compliant.

<h3>WooCommerce 2.6.14 Compatible</h3>

== Installation ==
Extract the plugin folder and copy the folder into your wp- content/plugins folder.

Log in to your WordPress admin account and click on the Plugins menu on the left.

You should see WooCommerce Forte Payment Plugin in the list of installed plugins. Click on the activate link to enable WordPress to use the Forte Plugin.

To configure the plugin with your details, click on WooCommerce Settings on the left menu.

Click on Payment Gateways tab from the top menu.

Click on Forte. You can configure the plugin according to your requirements.

Brief description of the fields:

Activate-Users can activate or deactivate Forte Payment Systems by choosing enable and disable button.

Title-The plugin will be displayed during checkout.

Description-Users will get a brief description of the payment gateway.

API Access ID-Users have to sign up at https://www.forte.net/pricing to get the API access ID.

Account ID-The plugin requires User ID.

Location ID- Users need to put their location ID.

Transaction Success Message- User will receive a transaction success message after a successful transaction.

Transaction Failed Message- User will receive a transaction failed message after a failed transaction.

Accepted Cards- User will have to select option from wide range of accepted cards.
